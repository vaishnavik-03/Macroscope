Macroscope is a health-focused web project that helps users understand
what and how much they eat by tracking **BMI, BMR, and calorie intake**.

## Features
- BMI Calculator
- BMR Calculator
- Simple food intake awareness
- Clean and beginner-friendly UI

## Tech Stack
- HTML
- CSS
- JavaScript

## Why this project?
Built as a learning project to explore health analytics and frontend logic.
